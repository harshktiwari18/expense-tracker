# Premium Expense Tracker 💰

Welcome to the **Premium Expense Tracker**, an elegant, modern, and beautifully designed web application built from the ground up using pure HTML, CSS, and Vanilla JavaScript. 

Take control of your personal finances with a solution engineered for simplicity and visual excellence.

## ✨ Key Features

- **Stunning UI/UX**: Immerse yourself in a carefully crafted "Dark Mode" aesthetic, accented with vibrant neons, sleek typography (`Inter` font), and glassmorphism elements.
- **Dedicated Landing & Authentication**: Features a full-screen dynamic landing page highlighting your financial journey, complete with a clean, centered login gateway.
- **Expansive Dashboard**: A widescreen dashboard view giving your data the breathing room it deserves. Visually distinguished lists automatically tag positive income (green) and negative expenses (red).
- **Instant Math**: Real-time recalculation of your total balance, income, and outgoing expenses right when you add or remove an item.
- **Indian Rupee (₹) Integration**: Default currency set to Local Indian formats, automatically displaying two decimal places for ultimate precision.
- **Local Persistence**: Built on local browser storage (`localStorage`); refresh the page entirely and your data stays right where you left it. Zero database setup required!

## 🚀 How to Run

1. Simply load the application by opening `index.html` in your favorite modern browser. 
2. **Login**: Put in any username and password combinations when accessing the login page—our initial prototype embraces an ultra-fast simulated authentication gateway perfectly suited for local testing.
3. **Start Tracking**: Dive right into the dashboard! Add new transactions via the form, review your net balance, or remove previous entries via the hidden hover-actions on your history list.

## 🛠 Tech Stack

- **HTML5**: Semantic tags mapped for SEO and accessibility.
- **CSS3 (Vanilla)**: Grid & Flexbox, comprehensive root variable theming, and rich keyframe/hover micro-animations.
- **JavaScript (Vanilla)**: Pure client-side interactions, Event Listening, DOM traversal and LocalStorage synchronization.
